# Bioinformatics
<p>Repository for student project</p>

# How to run project?
<p>Position yourself to bioinformatics/ directory.
Via command line insert:
make src

This command will output bwt* executable file.
Run command:
./bwt* <input_file> <output_file>
</p>

# How to run tests?
<p>Position yourself to bioinformatics/ directory.
Via command line insert:
make test</p>
