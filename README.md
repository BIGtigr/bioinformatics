# Bioinformatics
<p>Repository for student project</p>

# How to run project?
<p>Position yourself to bioinformatics/ directory. Run command:</p>
<p>make src</p>

<p>This command will output bwt* executable file. Run command:</p>
<p>./bwt* input_file output_file</p>

# How to run tests?
<p>Position yourself to bioinformatics/ directory. Run command:</p>
<p>make test</p>
